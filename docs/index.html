Lưu ý khi cài đặt

  • Kết nối Internet: Cần thiết để tải file.
  • Phiên bản Zalo: Cần cập nhật Zalo mới nhất (Tải Zalo).
  • Cài đặt: Chạy file .exe > Chọn "OK".
  • Gỡ cài đặt: Chạy file .exe > "OK" > "No" để gỡ.
  • Cập nhật: Chạy file .exe > "OK" > "Yes" để kiểm tra và cập nhật.
  • Hướng dẫn: Xem file "Zablind Instructions" ngoài Desktop sau khi cài.
Lưu ý bản Beta
Nếu phím Application/Menu không hoạt động, hãy thử Shift + F10. Nếu vẫn không được, hãy dùng phần mềm SharpKeys để gán lại phím.
